# Daily Link Tracker

A lightweight, mobile-first dashboard designed to manage your daily web routines. This app runs entirely in your browser with a strict emphasis on performance, data ownership, and privacy.

---

## Privacy & Security First

* **Zero Tracking:** No tracking cookies, no telemetry, no analytics, and no third-party scripts.
* **100% Offline / Local:** Requires no server or backend database.
* **Your Data is Yours Only:** Your link links, custom schedules, and click history never leave your device. The only external service is the DuckDuckGo favicon API.

---

## Features

* **Auto-Reset System:** Tracks the exact timestamp of your activity and automatically unchecks your checklist when your custom link reset time passes (e.g., midnight, evening resets, etc.).
* **Edit Mode:** Keep your interface clean. Toggle "Edit Mode" to manage your list, then hide it for a streamlined daily experience.
* **Drag-and-Drop Sorting:** Prioritize your link rotation order effortlessly.
* **Inline Profile Customization:** Add custom link names, their URLs, and configure individualized reset windows.
* **Dynamic Favicons:** Automatically pulls high-resolution icons safely via the DuckDuckGo favicon API.
